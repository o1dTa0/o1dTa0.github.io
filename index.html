辩论笔记：庞颖的反驳入门教学

Posted on 2018-10-29

庞颖：反驳入门

  • 分类：
    1. “杀人“：具有效果的出彩语言
    2. “推塔“：实质的论点进步
  • 立论回顾：
    1. 定义、范围、比较标准：思考有没有共识、优先反驳
    2. 论点：论证逻辑、例子、类比、数据、结论：拆哪都行、都拆也行

反驳定义、范围

  • 反驳的基本理念：有破有立、他为什么不对、我为什么正确
  • 定义反驳：
    1. 诉诸权威：词典、论文、专家
    2. 构建语境：用生动形象的大量例子
  • 范围反驳：
    1. 类似定义的范围（词义框定的范围）：还是得有根据、语境
    2. 讨论的范围：在题目没有规定限定到某一小范围时，通常给出更全面的一方占优。

反驳比较标准

第一类：大的标准有共识、但每个论点都其实是具体的标准

* 例子：延长退休年龄有利于老年人的物质和精神生活
    整体利弊没问题，关键在于论点的具体标准和成立与否
* 要有**层次概念**：论点和重要性分两个层次阐述
    1. 这个交锋点为什么是他错我对
    2. 这个交锋点为什么（对评胜负）很重要
    or为什么对方占优势的点并没什么用

第二类：大的比较标准说不清、简直就是两道辩题

*     例子：法海VS白素贞：佛法为先vs文学至上
* 第一种：**对方的比较标准比不出来**，完成不了比较标准的功能性
“拆不拆白素贞都有艺术价值、您方比较标准判断比较不出来”
**诉诸逻辑**

个人想法：这里的例子不是在说这个比较标准本身不合格（在这个比较标准之下胜出的一方当然得证）而是在说这个比较标准与论点（论题）不匹配，双方都都符合这个标准的筛选，没办法经由这个标准判断胜负
我觉得这也只能作为一种对于标准的优劣判断和削弱，因为这涉及到之后的交锋，双方论点都比不出来时至少是一个可辩性的体现，而且可以继续分析扩种、继续比较，未必比不出来

* 第二种：**对方的比较标准没有意义／不合常理／不合语境／霸道／不择手段**
**诉诸感性**

反驳论证

  • 需要反驳的论证模型：因为A->B，所以结论
    因为小王学习成绩好，所以小王应该当班长
  1. A带来B：小王成绩不好
  2. C也带来B：小李成绩也好（您需要给出其他理由进一步区分）
  3. B不重要：成绩好没用，班长需要的是领袖才能
  • 三种反驳途径不需要全讲，看力度和难易
  • 反驳逻辑之中还需要有干货：例子、数据、常识
  • 在准备辩论的时候根据前面的反驳思路验证补全自己的论点和反驳

    常见的反驳

  • 比偏概全／轻率的归纳
    发现：“一定、都、绝对”的应用
    反驳：举反例、追论证
  • 质疑对方根本没论证、逻辑不完整
    多想想为什么、凭什么、那又怎样、真的吗？
    背后是不是有什么错误的前提假设
    你甚至可以质疑一切
    要对质疑的东西提出相反的可能性（用干货）
    没有干活比较像找茬（杠精？）
  • 循环论证：用定义解释论证
    反驳：理清逻辑，告诉大家这为什么是一个循环论证／霸道定义
  • 复合原因：
    反驳：有破有立，指出背后的真正原因是什么（给出干货）
  • 反驳数据：
    1. 是否利益相关
    2. 年份有无、数据有无更新
    3. 相关性不证明因果
    4. 绝对值VS比较（变化）
    5. 是不是稳定状态（波动还是平稳）
    6. 调查方法、问卷设计、场景有无问题
    7. 调查对象是否有问题
  • 三段论：大前提、小前提、结论
  • 充分／必要条件：
    充分：有A一定有B，A是B的充分条件
    必要：没有A就不会有B，A是B的必要条件
    练习：B存在是否能预测A的存在，没有A是否一定没有B
    反驳：通过简单的类比举例说清楚逻辑
  • 其他：扣帽子（预设失败）、虚假两难、类比不当

反驳的基本格式

  1. 对方辩友说了XXX：避免曲解和扣帽子
  2. 说白了对方其实是这个意思：帮助评委明白逻辑
  3. 他为什么是错的
  4. 对的应该是什么：有破有立，帮助立论；增加对方的论证责任

一些反驳方法

  1. “按照对方辩友的逻辑”会推出不合常理／自相矛盾的荒谬结论
  2. “什么在您方看来是（我方成立的定义、例子、标准）”，您方自己的标准都不适用于您自己的例子，可见你的标准有问题（标准比较不出来）
    先达成共识（对方逃不了），再进行反驳

辩论笔记：庞颖的立论入门教学

Posted on 2018-10-29

庞颖：立论入门

定义

  1. 诉诸权威：
    • 百度百科、维基百科、新华字典
    • 书籍、论文
  2. 创造语境：
    • 老友赛：新国大对马大（人必须长大是好事／坏事）
    • 讲故事、讲例子
    • 做类比：同一定义在其他例子中的体现
    • 定义战时语境要多套（种说法）
  3. 公允但吃亏vs霸道但容易论证
    • 亚太赛：同化政策是否有利于国民团结
    • 个人风格问题

什么是好的立论

  • 是什么
    1. 论证逻辑线要完整
    2. 强有力的例子和数据
    3. 有深度、有后招
    4. 有讨论的意义
    5. 令对方措手不及
  • 怎么办
    1. 逻辑可以学
    2. 利用工具书
      • 平时好读书
      • 赛前利用好资料：论文、电子书
    3. 熟悉经典思辨问题
      • 道德、生命、权利、自由、民主、集体、工具、法律
      • 杀一救百是不是正义之举
      • 安乐死符不符合人道主义精神
      • 权利天赋vs权利人赋
      • 人有没有自甘堕落的自由
      • 尽如人意vs无愧于心；年轻人是不是应该为自己而活
      • 恶法依法
      • 找一场关于经典思辨的辩论赛、继而扩充了解立论反驳中的内容

比较标准

  • 转化为利大于弊（弊大于利）
    “王者荣耀是荣耀还是毒药“
  • 利弊怎么比较？
    1. 利的不可替代性和弊的可规避性
    2. 把利弊数字化
    3. 有／无助于：
      作为反方主动承担论证责任提出弊端
      量变促成质变才能产生实际意义
    4. “更应该靠谁“：能力+意愿
    5. “人生悲喜”：渲染 or 我们怎么看待观点、继而怎么选择
      不要太依赖价值，因为价值只能依靠渲染，不好操作且难分胜负；
      在虚中能打得更实会更有优势

论证

  1. 容易出现的问题：
    • 说了半天没有论证立场
      很多人喜欢说很多正确的话
      但是和需要论证的立场没什么关系
    • 逻辑跳跃，核心环节缺少论证
    • 太浅，容易被对方用深一层压制
    • 切入点太怪，不符合评审期待
  • tip：政策辩论其实到最后其实还是利弊比较（需根不容易出问题）
